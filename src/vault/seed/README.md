# ntropy vault

This directory is a note vault managed by
[ntropy](https://ntropy.westhoffswelt.de), a Markdown note-taking and
management CLI where metadata, not folders, is the filing system.

- `all-notes/` holds the notes: plain Markdown files with YAML frontmatter,
  named `<ulid>-<slug>.md`. This is the source of truth.
- The other top-level directories (such as `by-tag/`) are materialized views:
  symlinks into `all-notes/`, derived from the notes' frontmatter and
  regenerated by ntropy.
- `.ntropy/` holds the vault's configuration and templates.

## Working with this vault

Install the CLI:

```bash
cargo install ntropy
```

Then, from anywhere inside this directory:

```bash
ntropy list          # browse all notes
ntropy new "Title"   # create a note
ntropy search word   # filter, query, or full-text search
ntropy info          # show the vault and its statistics
```

See <https://ntropy.westhoffswelt.de> for the full documentation.
